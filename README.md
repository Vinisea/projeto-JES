# Projeto Jogos Estudantis SESI

## O que é?
Esse projeto foi feito para facilitar a vida de professores e organizadores do evento, centralizando todas as atividades dentro de um único web app.

## Qual é a vantagem?
Não será preciso fazer tudo manualmente, como a escalação dos times, chaveamento e placar final. Basta apenas dar alguns cliques.

## Tecnologias usadas: 
O projeto foi desenvolvido utilizando as seguintes tecnologias:
 - [**REACT**](https://www.react.dev)
 - [**JavaScript**](https://www.oracle.com/br/developer/javascript/)
 - [**CSS**](https://developer.mozilla.org/pt-BR/docs/Web/CSS)
 - [**Vite**](https://www.vite.dev)
 - [**NextJS**](https://www.nextjs.org)
